# 지훈 SMP 플러그인

**Purpur/Paper 1.20.4용 커스텀 플러그인**

## 기능

### 1. TPA (Teleport Ask)
- `/tpa <플레이어명>` - 텔레포트 요청
- `/tpaccept <플레이어명>` - 요청 수락
- `/tpadeny <플레이어명>` - 요청 거절
- 60초 후 자동 취소
- **비용 없음, 거리 제한 없음**

### 2. 위치 추적
- `/locate <플레이어명>` - 상대 위치 확인
- **비용**: 다이아 1개
- **표시**: 3초 동안 액션바로 좌표 표시

### 3. 경제 시스템
- **초기 코인**: 100원
- **킬 보상**: 300원
- **명령어**: `/coins` - 코인 확인

### 4. 강화 시스템
- **무기 강화**: 날카 10까지
- **방어구 강화**: 보호 10까지
- **비용**: 100원 / 강화
- **확률**: 초기 90%, 1강마다 10%씩 감소, 10강은 1%
  - 1강: 90%
  - 2강: 80%
  - 3강: 70%
  - ...
  - 9강: 10%
  - 10강: 1%

### 5. 팀 시스템
- `/team <RED|BLUE>` - 팀 선택
- `/myteam` - 현재 팀 확인

### 6. 킬 카운트
- `/kills` - 킬 수 확인
- 플레이어별 킬 수 기록

## 설치

### 필요 사항
- Java 17 이상
- Maven
- Purpur 1.20.4 서버

### 빌드
```bash
mvn clean package
```

### 설치
1. `target/jihun-smp-1.0.0.jar` 파일을 서버의 `plugins/` 폴더에 복사
2. 서버 재시작
